## Common Interview Questions

Q1: What is DevOps?\
A: DevOps is a culture that combines development and operations to automate and improve software delivery.

Q2: What is CI/CD?\
A: CI/CD automates code integration, testing, and deployment to ensure faster and reliable releases.

Q3: Jenkins vs GitHub Actions?\
- Jenkins: External tool, more control
- GitHub Actions: GitHub-native, simpler setup

Q4: Why automation is important in DevOps?\
A: Reduces human error, speeds delivery, improves consistency.
